## 🚀 Vision
**Substrate** is not just another note-taking tool. It is a **Learning Operating System** designed for senior engineers and researchers in AI, CG, Quant, and Web3. 

Instead of building MVPs with fragmented tutorials, Substrate helps you reverse-engineer your "Great Ideas" into "Fundamental Knowledge Paths." We don't build the product for you; we build the **capability** in you to build the product.

### 🧠 Core Components
1. **Idea Vault**: Transform vague inspirations into structured functional modules and capability requirements.
2. **Study Core**: Map capability gaps to specific chapters of classic textbooks (Math, OS, CS Theory) with AI-generated exercises.
3. **Knowledge Pack Platform (Future)**: A marketplace for domain experts to share high-quality, structured learning paths.

### 🛠 The "Make it Fast" Roadmap
- **Phase 1: Make it Work** (Current): MVP for idea capture and basic capability mapping.
- **Phase 2: Make it Right**: Refactor with clean architecture, robust DB schema, and professional engineering standards.
- **Phase 3: Make it Fast**: Deep dive into OS primitives, algorithm optimization, and high-performance execution.

### 💻 Tech Stack (Plan)
- **Backend**: Python (FastAPI) + SQLite
- **Frontend**: React + Vite (Tauri for Desktop)
- **AI**: Claude 3.5 Sonnet / GPT-4o Integration

## 🚀 愿景（中文）
**Substrate** 是一个面向 AI、CG、量化、Web3 等硬核工程领域的「**个人能力升级与学习操作系统**」。

我们不主张通过碎片化的教程快速拼凑 MVP，而是主张从你的“伟大想法”出发，反向推导出实现该想法所需的“基础学科路径”。LearnOS 不替你造产品，而是让你具备造产品的**能力**。

### 🧠 三大核心组件
1. **想法系统 (Idea Vault)**：把模糊的灵感转化为结构化的功能模块和能力需求。
2. **学习系统 (Study Core)**：将能力缺口映射到经典教材（数学、操作系统、计算理论）的具体章节，并生成针对性练习。
3. **知识包平台 (未来扩展)**：让领域专家贡献结构化的深度学习路线和实战题库。

### 🛠 三步走路线图
- **第一阶段：跑通流程 (Make it Work)**：实现想法收集与基础能力地图的最小闭环。
- **第二阶段：做对架构 (Make it Right)**：应用整洁架构、数据库范式和专业工程标准进行重构。
- **第三阶段：优化性能 (Make it Fast)**：深入操作系统底层，进行算法优化与高性能实现。
